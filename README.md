# Mool Taxi 🚖  

Welcome to the official documentation for **Mool Taxi**!  
Mool Taxi is an Android app designed to help taxi drivers track rides in real time, calculate fares automatically, and manage trip information efficiently.  

---

## 📱 Features  

- 🗺️ **Real-Time Map Tracking** – Integration with Google Maps API to display the driver’s live location and track movement during rides.  
- ⏱️ **Taxi Meter System** – Shows ride duration, distance traveled, and dynamically calculates fares. Start and stop rides with a single button.  
- 🔐 **Authentication System** – Secure registration, login, and password reset powered by Firebase Authentication.  
- 📍 **Location Permissions** – Requests and manages runtime location access using EasyPermissions.  
- 🔔 **Trip Summary Notifications** – Shows ride duration, distance, and final fare automatically at the end of each trip.  
- 👤 **Driver Profile** – Displays driver info, profile picture, and a QR code generated with QRCodeWriter. Logout securely.  
- 💾 **Local Data Storage** – Stores ride data locally using SQLite for reliability and offline access.  

---

## 🛠️ Technologies Used  

- **Kotlin**  
- **Google Maps API**  
- **Firebase Authentication**  
- **SQLite**  
- **EasyPermissions**  
- **QRCodeWriter**  
- **NotificationsManager**  
- **Android Studio**
